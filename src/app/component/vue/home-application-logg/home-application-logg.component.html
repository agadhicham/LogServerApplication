C'est quoi un fichier logg

Written by AL AGAD Hicham on 30 Mai 2019. Posted in github

Il s’agit d’un fichier comprenant différentes informations liées à l’utilisation d’un serveur, d’une application, d’un logiciel ou d’un système informatique. Un fichier log peut contenir certaines données confidentielles sur l’utilisateur.

En anglais log désigne le journal de bord du commandant d’un navire. Le terme a ensuite été repris en informatique.Il définit alors un historique lié à l’utilisation d’un serveur, d’un logiciel, d’une application et/ou d’un système. Dans la plupart des cas il s’agit d’un fichier texte dans lequel les évènements vont être classés de façon chronologique et ligne par ligne.Lorsque le fichier log est celui d’un serveur internet, il va confiner des informations comme la date et l’heure de connexion, mais également l’adresse IP des visiteurs, le système d’exploitation utilisé, le navigateur, les erreurs rencontrées, etc.Ces fichiers logs sont notamment utiles pour mesurer l’audience d’un site et obtenir des informations plus précises également sur le fonctionnement du serveur.

Nos rélisation

Written by AL AGAD Hicham on 30 Mai 2019. Posted in github

Notre politique de notre solution

Les données log de votre serveur Web révèlent toutes sortes de choses sur l’arrière-plan et le comportement de vos visiteurs. Ainsi une inspection des fichiers log (que l’on nomme aussi fichiers de traces ou fichiers journaux) peut vous apporter de nombreuses informations. Par exemple le navigateur utilisé par l’utilisateur, le temps passé par ce dernier sur votre site Internet ou encore le nombre de pages qu’il a consulté et enfin le moteur de recherche ou le lien qui lui a permis d’accéder à votre projet Web. C’est notamment pour cela que le fichier log est une source d’informations précieuse pour vérifier l’utilisabilité de votre site et pour l’optimisation d’un projet Web. Il est cependant impossible d’analyser manuellement ces données qui sont bien trop volumineuses. Toutefois, Il existe plusieurs outils d’analyse de fichiers log ou analyseurs de fichiers log qui peuvent réaliser cette tâche et présenter les résultats avec des visuels et des graphiques. Un représentant emblématique de ces analyseurs est l’outil open source GoAccess.
